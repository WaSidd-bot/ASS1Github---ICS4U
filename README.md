This is the mainframe for a website dubbed "The Infinite Online Dictionary."

this is a website that the user can search or put words into, and a java algorthm will compile
all possible defintions of that word included from multiple sources like MerriamWebster and Encyclopedia Britannica.

It will display a summary of all the accepted defintions, use cases, and the history and etymology of the word throughout time
in bite-sized chunks, easy to digest by the reader.

every single piece of information about the particular word will be available upon the users request, with links to all sources cited.

Enjoy!
